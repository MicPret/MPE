# MPE
MPE is a realtime 3D graphics engine powered by the OpenGL API.
***
## Hello, World!
The ```main.cpp``` file includes a simple first-person demo.
* Camera:
	* W: Forward
	* S: Backwards
	* A: Left
	* D: Right
	* Q: Up
	* E: Down
	* J, L: Rotate on the Y axis
	* I, K: Rotate on the X axis
* Shaders:
    * default: Blinn-Phong
	* U: Unlit
	* P: Phong
